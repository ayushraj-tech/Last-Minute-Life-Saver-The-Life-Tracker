# 📌 Overview

Managing multiple deadlines can be overwhelming. Traditional reminder applications simply notify users when it's already too late to plan effectively.

**Last Minute Life Saver** is designed as an intelligent productivity companion that proactively helps users:

* Prioritize important work
* Plan daily schedules
* Stay focused
* Track goals and habits
* Receive personalized AI recommendations
* Complete tasks before deadlines

The platform combines modern UI/UX with AI-powered productivity features to deliver a seamless and engaging user experience.

# 🚀 Getting Started

## Prerequisites

Before running the project, install:

* Node.js (v22+ recommended)
* npm
* Git
* Google Gemini API Key

---

# 📥 Installation

Clone the repository

```bash
git clone <repository-url>
```

Navigate to the project

```bash
cd last-minute-life-saver
```

Install dependencies

```bash
npm install
```

---

# 🔑 Environment Variables

Create a `.env.local` file in the project root.

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

---

# ▶️ Run the Application

Development

```bash
npm run dev
```

Production Build

```bash
npm run build
```

Preview Production

```bash
npm run preview
```

The application will be available at

```
http://localhost:3000
```

*(or the port shown in your terminal if different).*
